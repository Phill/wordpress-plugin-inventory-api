== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.

== Assumptions ==

In order to use this plugin you must have an active account with DealerTrend, Inc.
The plugin utalizes their API to pull automotive data.

It is assumed that people using this plugins have the necessary credentials to gain access to their feeds and the DealerTrend API.

== Configuration ==

To configure the plugin, go to the DealerTrend API settings page, make sure you are on the settings tab.

The feeds tab will show you the status of all your feeds based on your configuration of the plugin.

If you want to display your inventory feed on your WordPress website, enter in the Vehicle Managment System API address provided to you by DealerTrend.
You will then also need to enter in your company ID which should also be provided from DealerTrend.

If you wish to display virtual inventory or research data, enter in the Vehicle Reference System address provided to you by DealerTrend.
Click save.

After saving it will take you back to the feeds tab and let you know the status of each feed now that you have configured your site.

To change the look of your inventory list/detail page, use the theme tab. There you can also dictate how many vehicles you want to display on a listing page by default.

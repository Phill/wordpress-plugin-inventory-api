Shortcode Parameters

Place [inventory_list] in the content of a page to display inventory. e.g. [inventory_list saleclass=Used make=Ford model=F-150 tag=special]
Parameters Acceptable Values
saleclass New,UsedDefault: New
make Any Make Name (e.g. Dodge, Toyota, etc..)Default: All
model Any Model Name (e.g. Charger, Camry, etc..)Default: All
trim Any Trim Name (e.g. SE, XLE, etc..)Default: All
vehicleclass car, truck, sport_utility, van, minivanDefault: All
price_from Any numeric value (e.g. 1000)Default: empty
price_to Any numeric value (e.g. 100000)Default: empty
certified yesDefault: empty
tag special, gas-saver, cherry-deal, good-buy, low-miles, one-owner, sale-pending, custom-wheels, hybrid, local-trade-in, moon-roof, navigation, priced-to-go, rare, under-blue-book, wont-last *custom tags also acceptable Default: empty
limit Any numeric value (e.g. 20)Default: 10
style newspaper,newspaper_sb,clearDefault: newspaper
